# custom-desktop-timer
自分用のデスクトップタイマー

【設計メモ】
・Python + CustomTkinter(使ったことない、調べてみる)

・最前面に常駐するデスクトップアプリ
・(終了はタスクバーの通知領域から行えると嬉しい→pystray必要の可能性あり？、ver1.0では実装しない)
・ウィンドウサイズ・位置は固定

・10, 15, 20, 25分をワンクリックで計測可能
　→計測中は終了ボタンのみに
・残り時間を表示
・タイマー計測中は分かりやすく色を変えたりする
・終了時にサウンドで通知、ver1.0では標準音
・(終了時に休憩時間タイマー自動開始のオプション(ポモドーロ)、ver1.0では実装しない)
