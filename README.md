# custom-desktop-timer
自分用の常駐デスクトップタイマー
## 【基本操作】

・5~25分まで5分刻みのボタンを押すことでそれぞれ計測可能

・計測中にカウントされている時間の部分をクリックすると途中で終了

・00:00にはWindowsの標準音が3回鳴る

・タスクトレイから表示・終了・モニター切り替えが可能
